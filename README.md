\# OBI 2021 – Idade de Camila



\## Descrição da questão

Dado três idades diferentes, descobrir qual é a idade intermediária (nem a maior, nem a menor).



\*\*Exemplo:\*\*

\- Entrada: 15, 20, 18

\- Saída: 18



\## Estratégia utilizada

Somamos as três idades e subtraímos a maior (max) e a menor (min).

O valor restante é exatamente a idade intermediária de Camila.



\## Como executar

1\. Tenha o Python instalado

2\. Execute no terminal: `python solucao.py`

3\. Digite as três idades, uma por linha
